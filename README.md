# 🔥 Flames

A beautiful and interactive web application built with vanilla HTML, CSS, and JavaScript.

## Overview

Flames is a modern, responsive web project that showcases creative front-end development. The project features a clean interface with smooth interactions and elegant styling.

**Live Demo:** [https://sabithsuriya.github.io/flames/](https://sabithsuriya.github.io/flames/)

## Features

- 🎨 Beautiful and modern UI design
- ⚡ Fast and lightweight (pure vanilla JavaScript)
- 📱 Responsive design
- 🎯 Interactive user experience
- 🚀 Deployed on GitHub Pages

## Getting Started

### Prerequisites
- A modern web browser

### Local Setup

1. Clone the repository:
```bash
git clone https://github.com/sabithsuriya/flames.git
```

2. Navigate to the project directory:
```bash
cd flames
```

3. Open `index.html` in your web browser:
```bash
open index.html
```

Or use a local server:
```bash
python -m http.server 8000
# Then visit http://localhost:8000
```
